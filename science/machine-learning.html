  • MBA with Artificial Intelligence Specialism: an online degree by Coventry University that consists of eleven courses starting in September 2018.
  • Big Data: Statistical Inference and Machine Learning: online courses by Queensland University of Technology (QUT) hosted on FutureLearn. It is a three-week course that requires 2 hours of study per week. It requires tools such as R, R Studio, H20 Flow and Weka. The course is part of the Big Data Analytics program.
  • Statistics and R is an online course created by Harvard University that is hosted on edX. It is described as An introduction to basic statistical concepts and R programming skills necessary for analyzing data in the life sciences. It is a four-week course that requires two to four hours of study per week. The course is part of the Data Analysis for Life Sciences series.
  • Advanced Machine Learning: online course by The Open University (UK) and Persontyle that is hosted on FutureLearn. It is a four-week course that requires four hours of study per week. Some previous experience with machine learning and statistical modeling is expected. The course uses R.
  • Python Basics for Data Science is an online course created by IBM that is hosted on edX. It is a one-week course that requires 2 to 5 hours of study.
  • Machine Learning with Python: A Practical Introduction is an online course created by IBM and hosted on edX. It is a two-week course that requires 3 to 6 hours of study per week. The course requires some previous experience with Python and data science, e.g. through the course “Python Basics for Data Science”.
  • Deep Learning Fundamentals with Keras is an online course created by IBM that is hosted on edX. It is a five-week course that requires two to four hours of study per week. Keras is an open-source neural network library written in Python.
  • Artificial Intelligence (AI) is an online course created by Columbia University and is hosted on edX. It is a twelve-week course that requires eight to ten hours of study per week and expects previous experience with Python and an understanding of probability, linear algebra and calculus.
  • Data Science: Machine Learning is an online course created by Harvard University that is hosted on edX. It is a four-week course that requires two to four hours of study per week. It is part of Harvard University's Professional Certificate Program in Data Science and assumes that learners have taken the preceding (seven) courses in the program.
  • Machine Learning Foundations: A Case Study Approach is an online course created by the University of Washington and is hosted on Coursera. Is is the first course in the Machine Learning Specialization and requires five to eight hours of study per week over a period of six weeks.
  • Machine Learning is an online course by Andrew Ng (professor at Stanford University and co-founder of Coursera) that is hosted on Coursera. It consists of eleven sections that take between two and eight hours to complete (depending on the section). The course prerequisites are not described.
  • Neural Networks and Deep Learning is an online course created by deeplearning.ai and hosted on Coursera. It is the first course in a Deep Learning Specialization and requires three to six hours per week over a period of four weeks. The course requires basic Python programming skills, mathematics knowledge (matrix vector operations) and basic machine learning knowledge (how to frame a machine learning problem).
